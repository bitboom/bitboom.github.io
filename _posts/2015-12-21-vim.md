title: Default vim config
tags: [vim]
---

```
"  indent
set autoindent
set cindent
set smartindent
set backspace=indent,eol,start 

"  tab
set tabstop=4
set shiftwidth=4
set expandtab
set lcs=tab:..

"  highlight
set hlsearch
syntax on
filetype on

"  search
set ignorecase
set showmatch
set list

"  screen
set number
set ruler
set title
set background=dark

"  etc
set history=1000
```
